# NSAP Scheme Eligibility Prediction using Machine Learning

## 📌 Project Overview

This project predicts the most suitable National Social Assistance Programme (NSAP) scheme for an applicant using Machine Learning.

The model classifies applicants into different NSAP pension schemes based on demographic and socio-economic information.

This project was developed using IBM Watsonx.ai on IBM Cloud Lite as part of the Edunet Foundation – IBM SkillsBuild AICTE Internship.

---

## 🎯 Problem Statement

Predicting Eligibility for NSAP Schemes using Machine Learning.

The objective is to automate the identification of the correct pension scheme, reducing manual effort and improving the efficiency of government welfare distribution.
